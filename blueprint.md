
# Lotto Number Generator

## Overview

A simple, stylish web application to generate random lottery numbers. The application will feature a clean, modern interface and provide a delightful user experience with subtle animations and a premium feel.

## Project Outline

### **Design & Style**

*   **Layout:** Centered, single-column layout for mobile-first responsiveness.
*   **Color Palette:** A vibrant and energetic palette. Deep blue-gray background with bright, glowing colors for the numbers and buttons.
*   **Typography:** Expressive, bold fonts for headings and clear, legible font for the numbers.
*   **Effects:**
    *   **Background Texture:** A subtle noise texture on the main background for a tactile feel.
    *   **Depth:** Multi-layered drop shadows on the main container and interactive elements to create a sense of depth.
    *   **Glow Effect:** A "glow" effect on the generated numbers and the button to make them pop.
*   **Iconography:** A lottery or clover icon to enhance the theme.

### **Features**

*   **Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Display:** Displays the generated numbers in visually distinct circles.
*   **Interactivity:** A clear "Generate Numbers" button to trigger the action.
*   **Animation:** The numbers will animate into view when generated.

## Current Plan

1.  **Structure (`index.html`):** Set up the basic HTML structure with a main container, title, a placeholder for the numbers, and a generator button.
2.  **Styling (`style.css`):** Apply the modern design principles outlined above. This includes the layout, color scheme, fonts, shadows, and textures.
3.  **Logic (`main.js`):** Implement the JavaScript logic to generate and display the 6 unique lottery numbers when the button is clicked.
